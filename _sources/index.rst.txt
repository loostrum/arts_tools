Welcome to ARTS tools's documentation!
======================================
ARTS tools contains several script for interacting with Apertif Radio Transient System data.
This page contains an overview of the different functions available within the Python package.
For an overview of the functionality and typical usage of ARTS tools, refer to the project's
`Github page <https://github.com/loostrum/arts_tools>`_.
